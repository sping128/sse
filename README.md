# Senior Engineer Quest

A self-study web app for levelling up to senior software engineer. Built with vanilla HTML/JS — no build step, open files directly in a browser.

## How it works

- `index.html` — skill tree UI showing topics, XP, and unlock status
- `lesson.html` — lesson reader for studying each topic
- `lessons.js` — lesson content keyed by topic ID
- `progress.js` — single source of truth for all game state (XP, scores, topic status)

Study a lesson, take a quiz, update `progress.js` with the results. That's the loop.

## Topics covered

System design, design patterns, and senior engineering fundamentals — tracked as a skill tree with prerequisites and XP progression.
